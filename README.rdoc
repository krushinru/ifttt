== Getting started

1. Install the gem
  gem install ifttt-client
2. Sample usage:

  require 'ifttt'

  example=IFTTT::Profile.new("YOUR_WEBHOOK_KEY")
  example.get("EVENT_NAME") # Simple get requst
  example.post("EVENT_NAME",["array","of","values"]) # Post request with array of values
